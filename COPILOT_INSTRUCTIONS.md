# Copilot 指示事項 / 利用ガイド

このファイルは、このリポジトリで GitHub Copilot や自動化エージェント（以下「Copilot」）が作業する際の指示書です。
エージェントはここに書かれた方針に従い、安全かつ一貫性のある変更のみ行ってください。

---

## 1. プロジェクトの目的

- 小さな Raspberry Pi 向けの Python 時計アプリケーション（デモ・雛形）。
- 主な作業: ドキュメント整備、軽微なバグ修正、パッケージ設定の調整。

## 2. リポジトリ構成（参照）

- `src/rpi_clock/` — 本体ソース
- `pyproject.toml`, `requirements.txt`, `README.md`, `LICENSE`

## 3. 言語・実行環境

- サポート対象: Python 3.9〜3.11
- 開発時はユーザーが仮想環境を作ることを想定します（`.venv` は削除済み）。
- システム全体へのパッケージ書き換え（`sudo apt` 等）はユーザーの明示的な許可が無い限り行わないでください。

## 4. 変更方針（必須ルール）

1. 目的に関係ない大きなリファクタや設計変更は行わない（所有者の許可がある場合を除く）。
2. すべての Python 変更は対応するテストを追加するか、既存の動作に影響がないことを確認すること。
3. 依存関係を追加する場合は `requirements.txt` と `pyproject.toml` の両方を更新し、コミットメッセージに理由を記載すること。
4. 外部シークレット（API トークン等）をコードやコミットに埋め込まないこと。
5. `sudo` が必要な操作はユーザーの明示的許可なしに実行しないこと。

## 5. コーディングスタイル・品質

- 可読性を重視したシンプルなスタイルを優先します。
- `black` / `ruff` 等の導入は提案可。導入する場合は別コミットで行い、必要なら CI も更新すること。
- ドキュメント文字列は日本語で簡潔に記載して構いません。

## 6. テスト方針

- 現在このリポジトリには pytest ベースのテストは含まれていません。
- 将来テストを追加する場合は `tests/` にユニットテストを置き、`PYTHONPATH=src pytest` または仮想環境内で実行できるようにしてください。

## 7. CI（GitHub Actions）に関するルール

- 現在このリポジトリには GitHub Actions による自動 CI は含まれていません。
- 将来 CI を導入する場合は、事前に所有者に相談の上、ワークフローを追加してください。追加時は CI が通ることを必須とします。

## 8. Git / ブランチ / コミット

- デフォルトブランチは `main`。
- 小さな修正は直接 `main` に push して構いませんが、機能追加や破壊的変更はブランチを切り PR を経てマージしてください。
- コミットメッセージは英語か日本語で短く要点を記載してください（例: "Fix: remove markdown fences from __init__"）。

## 9. リモート操作・認証

- リモートは SSH（`git@github.com:nori-tama/rpi-clock.git`）を推奨します。HTTPS を使う場合は PAT を使用してください。
- Copilot がリモート操作（作成・push 等）を行う場合は必ず事前にユーザーの承認を得てください。

## 10. セキュリティとプライバシー

- ユーザーの個人情報やトークンをログやコミットに残さないでください。
- 外部ネットワークへ接続するコードを追加する場合は理由を明記し、レビューを受けてください。

## 11. 実行して良いタスクの例

- README の改善、軽微なバグ修正、`pyproject.toml` の整備
- 余分な Markdown フェンスやトークンの除去等のテキスト修正

## 12. 禁止事項

- ライセンスの無断変更、ユーザーの代わりに GitHub シークレットの追加や公開リポジトリ設定の変更を行うこと。

## 13. 作業ログの残し方

- 自動変更を行った場合はコミットメッセージと PR 説明に `Copilot: <短い説明>` を付け、変更理由とテストの有無を明記してください。

## 現状（2025-12-04）

- `tests/`（pytest）は削除済みです。
- GitHub Actions のワークフロー（`.github/workflows/ci.yml`）は削除済みで、空の `.github` ディレクトリも削除済みです。
- 最新の関連コミット: `9eecfde` — "Update Copilot instructions: reflect removal of tests and CI; add current status"
